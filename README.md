# Movie_booking_system
Using java and JDBC
This GitHub repository hosts a movie booking system implemented in Java Swing GUI, integrating MySQL database for storing movie data. The system allows users to browse available movies, check showtimes, book tickets, and manage bookings efficiently through a user-friendly graphical interface. The application employs object-oriented programming principles, utilizing classes such as `Movie`, `Showtime`, `Booking`, and `User`, with the main coordination handled by the `MovieBookingSystem` class. Users can contribute to the enhancement of features and usability, following established coding conventions and guidelines.
